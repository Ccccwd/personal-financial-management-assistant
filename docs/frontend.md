## 模块功能说明

### 1. 认证模块 (`src/views/auth`)
负责用户的登录与注册流程，保障系统安全性。
- **登录**：支持用户名/邮箱与密码登录，集成表单验证与错误反馈。
- **注册**：新用户注册入口，包含基本信息校验。
- **Token 管理**：登录成功后自动存储 JWT Token，并通过 HTTP拦截器自动处理请求认证。

### 2. 首页概览 (`src/views/home`)
系统的仪表盘，提供财务状况的即时视图。
- **资产总览**：实时显示当前总资产、本月收入与支出。
- **快捷操作**：提供“记一笔”等高频操作入口。
- **近期动态**：通过列表展示最近的几笔交易记录。

### 3. 交易管理 (`src/views/transaction`)
核心记账功能，支持全方位的交易记录管理。
- **新增交易**：支持收入、支出、转账三种类型的快速录入，包含分类选择、金额输入、备注及图片凭证上传。
- **交易列表**：按时间轴展示交易明细，支持按月筛选与查看详情。
- **交易详情**：展示单笔交易的完整信息，支持编辑与删除操作。

### 4. 账户管理 (`src/views/account`)
管理用户的多个资金账户（如现金、银行卡、微信、支付宝等）。
- **账户列表**：卡片式展示各账户余额、类型图标及资产占比。
- **账户维护**：支持新增账户、编辑账户信息（名称、余额、备注）及删除账户。
- **转账功能**：提供不同账户间的资金划转操作界面。

### 5. 统计分析 (`src/views/statistics`)
多维度的数据可视化报表，辅助用户分析消费习惯。
- **收支趋势**：通过折线图展示每日/每月的收支变化趋势。
- **分类占比**：利用饼图直观展示各分类支出的占比情况，支持点击查看详情。
- **对比分析**：提供收支对比柱状图，清晰展现财务健康度。

### 6. 预算管理 (`src/views/budget`)
协助用户进行财务规划与控制。
- **预算设置**：支持设置月度总预算及各分类的子预算。
- **进度监控**：实时展示预算使用进度条，超支时提供视觉预警。

### 7. 数据导入 (`src/views/import`)
支持外部账单数据的批量导入，简化记账流程。
- **微信账单导入**：支持解析微信导出的 CSV/XLSX 账单文件。
- **导入预览**：在正式入库前提供数据解析预览，支持用户确认与修正。

### 8. 设置中心 (`src/views/settings`)
用户个人偏好设置与系统配置。
- **个人信息**：查看与编辑用户头像、昵称等基本资料。
- **安全设置**：提供修改密码等账户安全功能。
- **分类管理**：支持自定义收支分类，满足个性化记账需求。

## 运行方式

### 环境要求

- Node.js >= 16.0.0
- NPM >= 8.0.0 或 Yarn / PNPM

### 1. 进入前端目录

```bash
cd frontend
```

### 2. 安装依赖

```bash
npm install
# 或者
yarn install
```

### 3. 配置开发环境

项目默认配置了开发代理，将 API 请求转发至后端服务器。
查看 `vite.config.ts` 中的 server 配置：

```typescript
server: {
  port: 3000, // 前端开发服务器端口，默认为 3000
  proxy: {
    '/api': {
      target: 'http://localhost:8000', // 后端服务器地址
      changeOrigin: true
    }
  }
}
```
确保后端服务已启动在 `http://localhost:8000`。

### 4. 启动开发服务器

```bash
npm run dev
```

启动成功后，访问控制台输出的地址（默认为 `http://localhost:3000`）。

### 5. 构建生产版本

```bash
npm run build
```

构建产物将输出至 `dist/` 目录，可部署至 Nginx 或静态文件服务器。
